Los formatos de salida son:
  • formato binario (PBF) formato
  • OSM XML, comprimido con gzip, bzip2 o 7-zip (xz)
  • o5m, comprimido con gzip o bzip2
  • Garmin formato, con estilos para OSM, mapa ciclo, OpenFietsMap, OpenSeaMap, OpenTopoMap, y ocio (freizeitkarte). El encaminamiento está habilitad. Note: some older devices don't support Unicode (UTF-8) and some new devices have a broken DRM firmware ("can not unlock map"). In this special case please use the ASCII style.
  • Esri shapefile formato
  • Osmand (OBF) formato para dispositivos Android
  • mapsforge formato para dispositivos Android
  • Navit formato para Linux, Windows, Android, OpenMoko Freerunner, WinCE, Nokia N800 tablas Internet, iPhone, Zaurus
  • libosmium OPL, GeoJSON, SQLite, text format
  • SRTM Höhenlinien für Garmin, Osmand, und PBF
  • BBBike Perl/Tk bbd format
